🧠 Why This Is Important

Searching directly in a database:
Client
   │
   ▼
Database
   │
LIKE '%laptop%'

This becomes slow as data grows.

Using Elasticsearch:
Client
   │
   ▼
Elasticsearch
   │
Indexed Documents
   │
Fast Search Results

Benefits:
	•	✅ Lightning-fast search
	•	✅ Full-text search
	•	✅ Fuzzy matching
	•	✅ Autocomplete
	•	✅ Ranked search results
